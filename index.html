Introducción a Python para Análisis de Datos

Python se ha convertido en uno de los lenguajes de programación mas populares en los ultimos años, debido en gran parte sus multiples aplicaciones: desde el analisis de de datos, automatizacion de procesos, hasta el desarrollo web, pasando por áreas como el aprendizaje automatico y las redes neuronales.

Bienvenido a este taller introductorio de Python para análisis de datos, en cual se darán las herramientas básicas necesarias para que inicie su desarrollo analista de datos con Python y en general como programador de Python (pythonista).
