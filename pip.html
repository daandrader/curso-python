Gestión de librerías

En programación un librería es una extensión del lenguaje original con alguna finalidad especifica, por ejemplo en python existen librerías especificas para analizar y gráficar datos, otras librerías para realizar análisis estadísticos y aprendizaje automático, librerías para desarrollo web entre otros. Algunas de las librerías que serán usadas en esta serie de talleres son: pandas, matplotlib, seaborn y numpy.

PIP

Para tal fin python cuenta con herramientas integradas para la gestión de librerías (instalar, desinstalar y actualizar) una de ellas es PIP, una herramienta que se usa mediante la consola o terminal.

Comandos básicos de PIP
  • Instalar librería
    • pip install NombreLibreria
  • Desinstalar librería
    • pip uninstall NombreLibreria
  • Actualizar librería
    • pip install --upgrade NombreLibreria

(Opcional) Anaconda

Anaconda es una distribución de Python con interfaz gráfica enfocada a ciencia de datos, donde no solo se incluye la instalación de Python sino una gran cantidad de librerías enfocadas al análisis, visualización de datos y editores entre otros. Cuando se instala Anaconda no es necesario usar PIP para la gestion de librerías.
