Introducción a Python para Análisis de Datos

Python se ha convertido en uno de los lenguajes de programacion mas populares en los ultimos anos, debido en gran parte a la multiples aplicaciones posibles: desde el analisis de de datos, hasta el desarrollo web, pasando por areas como el aprendizaje automatico y las redes neuronales.

Bienvenido a este taller introductorio de python para analisis de datos, en cual se daran las herramientas basicas necesarias para que inicie su desarrollo analista de datos con python y en general como programador de python (pythonista).
